| Versão         | Ano  | Principal Novidade                                               | Curiosidade                                                                                      |
|----------------|------|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| **Windows 1.0** | 1985 | Interface gráfica sobre o MS-DOS. O nome original do projeto era "Interface Manager". | Incluía o jogo *Reversi* para incentivar o uso do mouse.                                         |
| **Windows 2.0** | 1987 | Janelas sobrepostas, suporte a atalhos de teclado e primeiros usos do processador Intel 386. | Primeira versão que permitia usar o Excel e o Word com interface gráfica.                        |
| **Windows 3.x** | 1990 | Gerenciador de Programas e suporte a 256 cores. Popularizou o uso do mouse com o jogo *Paciencia*. | Foi o Windows que popularizou o PC no mercado doméstico e empresarial.                           |
| **Windows 95**  | 1995 | Introduziu o Menu Iniciar, Barra de Tarefas e Plug and Play.    | A Microsoft pagou milhões para usar a música "Start Me Up" dos Rolling Stones no comercial.       |
| **Windows 98**  | 1998 | Suporte nativo a USB, Active Desktop e melhor integração com internet. | A versão SE trouxe melhorias e estabilidade, sendo muito usada mesmo após o lançamento do ME.    |
| **Windows ME**  | 2000 | Introduziu a Restauração do Sistema e melhor suporte multimídia. | Conhecido por ser instável, foi uma das versões menos populares da Microsoft.                    |
| **Windows 2000**| 2000 | Base NT, focada em estabilidade e segurança para empresas.       | Foi base para versões de servidor e rede corporativa.                                           |
| **Windows XP**  | 2001 | Unificação das linhas 9x e NT, interface "Luna" colorida.        | O wallpaper “Bliss” é um dos mais famosos do mundo, uma foto real sem Photoshop.                 |
| **Windows Vista**| 2007| Interface Aero com transparências e busca integrada.             | Muitos usuários preferiram continuar no XP devido aos altos requisitos de hardware do Vista.     |
| **Windows 7**   | 2009 | Interface refinada, melhor desempenho e suporte a toque.         | Considerado um dos Windows mais estáveis e populares da história.                               |
| **Windows 8**   | 2012 | Interface Metro otimizada para toque, inicialmente sem Menu Iniciar. | A ausência do Menu Iniciar gerou forte rejeição, que levou à volta parcial no 8.1.               |
| **Windows 10**  | 2015 | Atualizações contínuas, integração com Cortana e Menu Iniciar híbrido. | Reintroduziu o Menu Iniciar com “Live Tiles” dinâmicos.                                        |
| **Windows 11**  | 2021 | Interface modernizada, ícones centralizados e foco em multitarefa. | Exige chip TPM 2.0 para segurança, uma novidade obrigatória e polêmica.                         |
